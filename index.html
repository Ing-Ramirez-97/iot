Proyecto IoT - Comunicación HMI y Nube

Este proyecto tiene como objetivo desarrollar un sistema de comunicación entre una pantalla HMI y la nube utilizando el protocolo MQTT. Se busca enviar información de variables como peso, presión, temperatura y horas para su posterior análisis y visualización.
